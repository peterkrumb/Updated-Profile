Case Study 1: API Project

Throughout the duration of my bootcamp, an integral part of the learning process are our major projects, of which we have 3. Our first one was an assignment testing our knowledge and grasp of APIs, an external database. For this project my partner and I decided to use the Giphy API as well as an API for basketball stats called BallDontLie. Originally we set out to use Youtube to render a highlight reel for a player when their name was typed in the search bar, but it proved too difficult given our limited span of time, so a Gif was rendered to the screen instead. For our other API we used an NBA data API which contained information dating back to 1979 which included player and team stats.
